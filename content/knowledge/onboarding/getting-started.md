## Where should I start if I have never used software like this?

Start with the practice setup flow. It walks you through adding your first product, step by step, so you can see how S.C.O.R.E. works before using it with your real business.

## Is S.C.O.R.E. hard to learn?

S.C.O.R.E. is built to feel simple, even if you've never used business software before. Most people learn the basics in just a few minutes.

## Setting up your first product

In the real setup flow, you'd add a product name, a starting quantity, and a price.

## Practicing a sale

Recording a practice sale walks you through the same few taps as a real one: pick a product, confirm the quantity, and complete the sale.
